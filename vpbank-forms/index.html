🔧 Hướng Dẫn Tích Hợp AI

Mỗi form đều expose JavaScript API để tích hợp với AI Service:

// Khởi tạo AI Form
const formAI = new FormAI('formId');

// Điền form tự động
formAI.fillForm({
  customerName: 'Nguyễn Văn An',
  loanAmount: '500000000',
  loanTerm: '24'
});

// Xác thực form
const validation = formAI.validateForm();

// Gửi form
await formAI.submitForm();

// Parse natural language (tiếng Việt)
const data = formAI.parseNaturalLanguage(
  "Điền khách hàng Nguyễn Văn An vay 500 triệu, kỳ hạn 24 tháng"
);
